## 宏观经济
#### PMI
**全称：**Purchasing Managers' Index  
**中文：**采购经理人指数  
**计算公式：**  
PMI = 订单权重×30%+产品生产权重×25%+就业情况权重×20%+厂商表现权重×15%+存货权重比例×10%  
**表达经济：**   
常以50%作为经济强弱的分界点： 
 
- 当指数高于50%时，则被解释为经济扩张的讯号。  
- 当指数低于50%，尤其是非常接近40%时，则有经济萧条的忧虑。  
-  一般在40~50之间时，说明制造业处于衰退，但整体经济还在扩张。
